# pdmn

Password Manager

- Rustだけ使う
    - Rustのメリットを最大限活かす
- Passwordを管理する
    - 自分にとって心地よいもの
        - 自分が一番のユーザ
        - コマンドライン
        - ファジーサーチ
    - 保存形式
        - データベース？
        - 暗号化 AES
        - 暗号形式はファイル名に
- Passwordを生成する
    - 強度の高い乱数
        - 暗号学的乱数
        - TRNG
    - パラメータ指定
        - 文字数、文字種、
- 最初はできるだけ小さく、動くものを作ること
